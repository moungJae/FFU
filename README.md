### 3/11 

+ 프로젝트 주제 결정

+ 주제 : 코로나 바이러스로 인해 대면 활동이 어려워진 사람들을 위한 친구 추천 APP

+ 프로젝트 설명 : 사용자의 정보(취미, 관심사, 성격, 나이, 활동 지역, ...)와 현재 위치를 기준으로 추천한다.

+ 핵심은 실제 사진을 애니메이션처럼 변환한 사진을 프로필로 설정하는 것이다.

+ 실제 사진이 아닌 애니메이션 사진으로 정함으로써 사용자의 익명성이 보장되며, 사진 도용에 대한 우려가 없다.
